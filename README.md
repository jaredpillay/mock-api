# Mock API

This project is a simple mock API built with Node.js. It is designed to simulate API endpoints for testing and development purposes.

## Features
- Simulates RESTful API endpoints
- Easy to configure and extend
- Lightweight and fast

## Prerequisites
- Node.js (v14 or higher)
- npm (Node Package Manager)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jaredpillay/mock-api.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mock-api
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the mock API server, run the following command:
```bash
npm run dev
```

The server will start and listen for requests. By default, it runs on `http://localhost:3000`.

## Project Structure
- `server.js`: The main entry point for the server.
- `package.json`: Contains project metadata and dependencies.
